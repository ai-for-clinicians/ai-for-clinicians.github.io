The dream of the semantic web

It's easy to forget how recent of an innovation the world wie web truly is, given how much dominance it has gained over how we conduct our daily lives. Tim Berners-Lee created the underlying hypertext format in 1989, while working at the research institute CERN, explicitly aiming to improve the way that researchers access and exchange information. While the basic idea was to give them a way to easily refer to other documents via links, to create a connected web of documents, ever since then, extension to the basic underlying paradigm of marked-up documents being shared over the internet have been proposed, but none have been as successful as the web itself. Tim Berners-Lee himself proposed one such extension in 1999, having realized that much of the existing content of the web was only suitable for the consumption by humans, not by machines. In his vision for what later became know as the semantic web, the British computer scientist aimed to make the world wide web machine-readable, putting automatic agents on the same footing as humans,
A machine-readable web where automatic agents are on the same footing as humans, as content is designed to be consumed by them. Linking data sources. Improved collaboration through annotation.
